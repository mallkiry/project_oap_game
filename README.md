# project_oap_game
> Проект по дисциплине ОП 03 Основы алгоритмизации и программирование
## Название: 
"Игра-виселица"
## Проблема, которую решает мой проект
В современном мире, который перенасыщен сложными графическими играми, существует потребность в простых, минималистичных и быстрых логических играх, не требующих мощных ресурсов компьютера. Тем самым простой, понятный интерфейс и игра, которая известна многим с детства, скрасят любое времяпрепровождение.

подростки и молодые люди

## Цели и задачи моего проекта:
- Оформить введение проекта
- Сделать Use Case диаграмму и блок схему
- Реализовать код по функционалу игры
- Реализовать код по оформлению игры
- Подготовиться к защите проекта
- Составить и оформить текстовую документацию в соотвествии с ГОСТ
- Составить и оформить визуальную презентацию проекта(презентация) в соотвествии с ГОСТ
## Функионал:
- [x] Реализация старта игры
- [x] Программа считывает символы(буквы), введенные с клавиатуры
- [x] Проверяет правильность введеного  
- [x] При правильном вводе, заполняет эти буквы в специально отведенные ячейки
- [x] При неправильном вводе, заполняет рисунок виселицы - подсчет ошибок
- [x] Записывает буквы, которые уже использованы
- [ ] Считывает и сохраняет количество побед
- [x] Реализация окончания игры
