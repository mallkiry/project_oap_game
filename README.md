# project_oap_game
> Проект по дисциплине ОП 03 Основы алгоритмизации и программирование
## Название: 
"Игра-виселица"
## Проблема, которую решает мой проект
В современном мире, который перенасыщен сложными графическими играми, существует потребность в простых, минималистичных и быстрых логических играх, не требующих мощных ресурсов компьютера. Тем самым простой, понятный интерфейс и игра, которая известна многим с детства, скрасят любое времяпрепровождение.
